# Explorer Contract
Status: Normative

## Responsibilities

- browse canonical `SCIR-H` and `SCIR-L`,
- navigate by module, symbol, stable ID, and provenance link,
- surface profile, preservation, tier, and boundary metadata,
- show H/L relation without inventing new semantics.

## Required views

- module dependency view
- symbol view
- effect and capability view
- boundary view
- provenance view
- H/L diff view

## Non-goals

- semantic inference beyond what the specs already define,
- hiding unsupported regions to make the graph look cleaner.
